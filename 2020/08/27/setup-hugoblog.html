什么是Hugo

引用一下Hugo官网的描述:

The world’s fastest framework for building websites.

Hugo是一个非常受欢迎的、开源的静态网站生成工具，和Hexo类似。 它速度快，扩展性强。

更多的关于Hugo的介绍，请参考Hugo的官网 https://gohugo.io/ 。

Hugo的安装和使用

建立 Hugo 项目

hugo new site hgblog

选择新的主题（Hugo maupassant）

git clone https://github.com/flysnow-org/maupassant-hugo themes/maupassant

Hugo maupassant 主题配置

可以先直接把 themes/maupassant/exampleSite 直接copy到项目根目录

cp -r themes/maupassant/exampleSite/ .*

添加新文章

hugo new post/my-first-blog.md

修改 post/my-first-blog.md

本地预览

hugo server

网页上打开 http://localhost:1313

生成静态文件

hugo

上传到站点

  1. MD 文件上传到本repo
    > git add .
    > git commit -m "comments here"
    > git push
  2. public 目录下文件上传到 mountain369.github.io
    > cd public
    > git init
    > git add .
    > git commit -m "comments here"
    > git remote add origin https://github.com/mountain369/mountain369.github.io.git
    > git push origin master
